# Auth0 MCP Server
# No external Python dependencies - uses only standard library
# Requires Python 3.7+

# Environment Variables Required:
# AUTH0_DOMAIN - Your Auth0 domain (e.g., your-tenant.auth0.com)
# AUTH0_CLIENT_ID - Management API client ID
# AUTH0_CLIENT_SECRET - Management API client secret
#
# Set up a Machine-to-Machine application in Auth0 Dashboard:
# 1. Go to Applications > Create Application
# 2. Choose "Machine to Machine Applications"
# 3. Authorize it for the Auth0 Management API
# 4. Grant required scopes (e.g., read:users, update:users, etc.)
